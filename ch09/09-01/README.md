# Adapter

## 디자인 원리

- 서로 다른 인터페이스를 연결함

- 이미 사용중이거나 정의된 인터페이스를 변경하지 않고 기존에 사용하던 대로 사용할 수 있도록 Adapter 객체를 제공

- 클라이언트가 사용하던 방식대로 호출하여 사용할 수 있도록 조정해줌

예) 안드로이드의 ListView를 만들때 사용되는 Adapter - 실제 Item 과 View를 연결해주는 기능
